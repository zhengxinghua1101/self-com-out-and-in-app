# 打鼾干预测试工具

基于 Electron + SerialPort 实现的跨平台串口测试工具，用于验证智能床垫/睡眠设备的打鼾检测与干预功能。

## 功能特性

- ✅ 串口自动检测和连接
- ✅ 自动加载内置鼾声数据文件
- ✅ 可配置采样率循环发送
- ✅ 实时计时显示（测试时长）
- ✅ 简洁易用的图形界面
- ✅ 支持 Windows/Mac/Linux 全平台

## ⚠️ 采样频率配置（重要）

### 当前配置

| 参数 | 值 | 说明 |
|------|-----|------|
| **采样频率** | **15 Hz** | 每秒发送 15 条数据 |
| **发送间隔** | **67 ms** | 1000ms ÷ 15 ≈ 67ms |
| **单轮数据** | 5001 条 | sn_5.txt 数据总量 |
| **单轮耗时** | 约 5分35秒 | 5001 × 67ms |
| **两轮耗时** | 约 11分钟 | 覆盖原程序 2-5 分钟触发窗口 |

### 修改采样率

编辑 `main.js` 第 118 行：

```javascript
// 公式：间隔(ms) = 1000 ÷ 采样频率(Hz)
sendInterval = setTimeout(sendLoop, 67);  // 15Hz = 67ms
```

常用参考值：

| 采样率 | 间隔 | 单轮耗时 | 适用场景 |
|--------|------|----------|----------|
| 1 Hz | 1000ms | 83分钟 | 慢速测试 |
| 5 Hz | 200ms | 16分钟 | 慢速验证 |
| **15 Hz** | **67ms** | **5分35秒** | **推荐默认** |
| 20 Hz | 50ms | 4分10秒 | 快速测试 |
| 30 Hz | 33ms | 2分45秒 | 超快速验证 |

### 选择建议

- **常规测试**：用 15Hz，5-10分钟触发干预
- **快速调试**：用 20-30Hz，2-5分钟触发
- **真实场景模拟**：用 1Hz，接近真实睡眠数据速率

## 使用说明

### 测试流程

1. **连接硬件**：将设备通过 USB 接入电脑
2. **选择串口**：从下拉列表选择设备串口，点击「打开串口」
3. **加载数据**：程序自动加载 `my_data/sn_5.txt`，也可点击「浏览」选择其他数据文件
4. **开始测试**：点击「开始测试」，程序循环发送数据
5. **观察结果**：人工判断打鼾干预是否触发（如振动电机启动）

### 数据文件格式

数据文件为纯文本格式，每行一条十六进制数据：

```
6D053A65F03CFA37
6D078D08F0EB4428
6D09C31BF171A359
...
```

## 常见问题

### Q: 串口打开失败？
- 检查设备是否正确连接
- 确认串口未被其他程序占用
- Windows 需安装对应串口驱动

### Q: 长时间不触发干预？
- 检查采样率设置是否过高
- 确认设备打鼾参数设置正确
- 尝试降低采样率，给算法足够检测时间

### Q: 如何更换测试数据？
- 将新的 txt 文件放入 `my_data` 目录
- 或点击「浏览」按钮直接选择文件
