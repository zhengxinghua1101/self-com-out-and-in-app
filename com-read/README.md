# 串口数据模拟工具

一个基于 Electron 的串口数据模拟读取工具，通过虚拟串口对接收数据并将数据导出为 TXT 文件。

## 功能特性

- 🎯 **虚拟串口对**：自动创建虚拟串口对，支持向写入端口发送数据
- 📥 **数据导出**：一键将接收的数据下载为 TXT 文件
- 📊 **实时显示**：数据实时滚动显示，自动记录时间戳
- 🧹 **数据管理**：支持清空数据，方便重新开始
- 🔄 **自动更新**：支持应用程序自动更新

## 平台兼容性

### macOS

- 依赖：需要安装 `socat`
- 安装方式：`brew install socat`

### Windows

- 依赖：需要安装 `com0com` 虚拟串口驱动
- 下载地址：https://sourceforge.net/projects/com0com/
- 安装后需重启电脑使驱动生效

## 安装运行

```bash
# 安装依赖
npm install

# 启动应用
npm start

# 打包应用 (macOS)
npm run build

# 打包应用 (Windows)
# 在 Windows 上运行：npm run build
```

## 使用说明

1. **设置波特率**：输入串口波特率（默认 9600）
2. **开始模拟**：点击"启动虚拟串口"按钮创建虚拟串口对
3. **发送数据**：向显示的"写入端口"发送数据（使用串口调试工具）
4. **查看数据**：接收的数据会实时显示在界面上，自动添加时间戳
5. **下载数据**：点击"下载数据"将所有数据保存为 TXT 文件
6. **清空数据**：点击"清空"按钮清除所有已接收的数据

## 虚拟串口对说明

启动成功后会创建两个虚拟串口：
- **读取端口**：应用程序自动连接此端口接收数据
- **写入端口**：使用其他串口工具连接此端口发送数据

### macOS 示例
- 读取端口：`/dev/ttys003`
- 写入端口：`/dev/ttys004`

### Windows 示例
- 读取端口：`CNCA0`（或 COM3 等）
- 写入端口：`CNCB0`（或 COM4 等）

## 注意事项

### macOS
- 首次使用需先安装 socat：`brew install socat`

### Windows
- 必须安装 com0com 虚拟串口驱动
- 安装 com0com 时需要管理员权限
- 安装后建议重启电脑
- 如果应用找不到串口，尝试手动指定 COM 端口号

### 通用
- 数据文件默认保存到下载目录
- 文件名自动按日期生成
- 关闭应用时会自动关闭串口和清理资源
